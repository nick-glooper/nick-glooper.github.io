Husian News Article Uploader

Upload and process news articles for the Husian website without using the command line.

How to use this tool:

  1. Create your article file following the required format (Title, Summary, Article).
  2. Save the file in the "News_Articles" folder or select it using the uploader below.
  3. Click "Process Article" to update the website with your content.
